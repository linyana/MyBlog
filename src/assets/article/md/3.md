# 自定义迭代器
## 理解迭代器底层原理
```javascript
class Counter {
    constructor(limit){
        this.limit = limit;
    }
    [Symbol.iterator](){
        let count = 1,limit = this.limit
        return {
            next(){
                if(count <= limit){
                    return {
                        done:false,
                        value: count++
                    }
                }else {
                    return {
                        done: true,
                        value: undefined
                    }
                }
            },
            return(){
                console.log('Exit early');
                return {done: 'true'}
            }
        }
    }
}
```
